# mapin-server

## 準備

- nodeとnpmをインストールしておく

## 起動方法

```
npm install
node index.json
```

## WebAPI

base url: https://mapin-server.herokuapp.com

- GET `/api/companies`
- GET `/api/companies/:company_id`
- GET `/api/shops`
- GET `/api/shops/:shop_id`
- GET `/api/shops/:shop_id/files`
- GET `/api/files`
- GET `/api/files/:file_id`
